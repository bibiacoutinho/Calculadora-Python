# Como instalar
```
git clone git@github.com:bibiacoutinho/Calculadora-Python.git
```
No diretório raiz:
```
virtualenv -p /usr/bin/python3 ENV
source ENV/bin/activate
pip3 install -r requirements.txt
```
# Como executar
No diretório raiz:
```
cd src
python3 app.py
```
Para desativar virtualenv:
```
deactivate
```
# Tecnologias utilizadas
* Python 3.6.9
* Pip 9.0.1
* Virtualenv 20.7.2
